# Stock Market Visualizer

## Overview

A Python-based application that analyzes and visualizes stock market data from a CSV file.

## Features

* Reads stock price data from CSV files
* Calculates average, maximum, and minimum stock prices
* Computes moving averages using NumPy
* Visualizes stock price trends using Matplotlib
* Displays price and moving average on the same graph

## Technologies Used

* Python
* NumPy
* Matplotlib
* CSV Module

## Project Structure

Stock_market_Visualizer | data | stock_data.csv | main.py

## Output

The program generates:

* Statistical analysis of stock prices
* Stock price trend graph
* Moving average visualization
